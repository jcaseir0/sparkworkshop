# Spark Workshop
Arquivos necessários para os laboratórios:

## 0_create_data.txt
Anexo o script para a criação do conjunto de dados. A extensão do arquivo deve ser alterada para .py.
Em seguida, execute o comando spark3-submit 0_create_data.py

## 1_demo-1.txt
A extensão do arquivo deve ser alterada para .py.

Execução:

Spark 2
```shell
spark-submit 1_demo-1.py /training/spark3/<<user_name>>/1_demo-1/spark2 /tmp/workshop/<<user_name>>/1_demo-1/spark2_explain.txt
```
Spark 3:
```shell
spark3-submit 1_demo-1.py /training/spark3/<<user_name>>/1_demo-1/spark3 /tmp/workshop/<<user_name>>/1_demo-1/spark3_explain.txt
```
